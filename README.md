# NuSen

Android app that displays local weather both now ("nu") and later ("sen").

## Has

* Fast boot, remembers location.
* Updates weather every 30 min.
* Can set later ("sen") to be between 1-6hrs.
* Can set relevant times (ex. 7:00 - 22:00)
